# Installation

Auf dieser Seite finden Sie Installationsanleitungen für die erforderliche Software.

Es gibt zwei Installationsprofile:

- **[Schnellinstallation](schnellinstallation.html)**: Umfangreiche Installationen für alle Fachschaften
- **[HMS Schnellinstallation](hms-schnellinstallation.html)**: Reduzierte Installationen nur für HMS

Beide Anleitungen bieten automatisierte Installationsskripte für Windows (PowerShell) und macOS (Homebrew), die den Prozess erheblich vereinfachen.
